**A Next-Generation Desktop Assistant for the Visually Impaired.**

Universal Access AI is an intelligent accessibility tool designed to help users with vision impairments navigate their digital and physical environments. By combining Computer Vision, Optical Character Recognition (OCR), and Generative AI, it acts as a "digital pair of eyes" that can see, read, and converse.
